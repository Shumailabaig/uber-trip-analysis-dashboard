# uber-trip-analysis-dashboard
# Uber Trip Data Analysis Dashboard (2016)

# Project Overview

This project analyzes Uber trip data from 2016 using **Excel and Power BI** to uncover insights about trip purposes, categories, mileage, and monthly usage patterns.

The dashboard provides interactive visualizations to explore travel behavior, business vs personal trips, and total distance traveled.

# Tools Used

* Microsoft Excel (Data Cleaning & Preparation)
* Power BI (Data Visualization & Dashboard)
* GitHub (Project Documentation & Version Control)
